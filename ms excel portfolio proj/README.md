# HR Analytics Dashboard

## Overview

This project analyzes employee data using Microsoft Excel to understand attrition trends and workforce demographics. An interactive dashboard was created to visualize key HR metrics and help identify factors affecting employee retention.

## Problem Statement

The goal of this project is to analyze employee data and understand:

* Employee attrition patterns
* Workforce demographics
* Factors that may influence employee turnover

## Process

### 1. Data Analysis

Employee data was analyzed using Excel.

### 2. Data Cleaning

The dataset was already mostly clean. Basic checks were performed for:

* Missing values
* Duplicate records
* Data consistency

### 3. Dashboard Creation

An Excel dashboard was built using Pivot Tables, Charts, and Slicers.

## Dashboard KPIs

* Total Employees
* Employees Left (Attrition)
* Active Employees
* Attrition Rate
* Average Age

## Dashboard Visualizations

* Employees by Gender
* Education-wise Attrition
* Attrition by Job Role
* Attrition by Department
* Attrition by Age Group
* Attrition by Marital Status

## Tools Used

* Microsoft Excel 2021
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Functions

## Outcome

The dashboard helps HR teams quickly identify attrition trends and understand employee demographics through interactive visualizations.
