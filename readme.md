# Bank Management System

A desktop banking application developed using Python and Tkinter that simulates core banking operations through a graphical user interface.

## Features

- Create New Account
- Secure PIN-Based Login
- Deposit Funds
- Withdraw Funds
- Check Account Details
- View Transaction History
- Input Validation and Error Handling

## Technologies Used

- Python
- Tkinter
- Object-Oriented Programming (OOP)
- Exception Handling

## Project Overview

This project demonstrates the implementation of a simple banking system where users can create accounts, authenticate using a PIN, perform transactions, and track transaction history.

The application was developed to strengthen practical understanding of:

- Functions
- Classes and Objects
- Constructors
- Event Handling
- GUI Development
- Transaction Processing

## How to Run

Clone the repository:

```bash
git clone https://github.com/your-username/Bank-Management-System.git
```

Navigate to the project folder:

```bash
cd Bank-Management-System
```

Run the application:

```bash
python bank-management.py
```
